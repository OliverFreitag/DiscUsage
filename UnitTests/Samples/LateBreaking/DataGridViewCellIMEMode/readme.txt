DataGridView Cell ImeMode Sample
================================
     This sample demonstrates how to control the ImeMode of each cell on the DataGridView control.


Sample Language Implementations
===============================
     This sample is available in the following language implementations:
     C#
     Visual Basic


To build the sample using the command prompt:
=============================================
     1. Open the Command Prompt window and navigate to one of the language-specific subdirectories under the DataGridViewCellIMEMode directory.
     2. Type msbuild DataGridViewCellIMEModeCS.sln or msbuild DataGridViewCellIMEModeVB.sln, depending on your choice of programming language.


To build the sample using Visual Studio:
=======================================
     1. Open Windows Explorer and navigate to one of the language-specific subdirectories under the DataGridViewCellIMEMode directory.
     2. Double-click the icon for the .sln (solution) file to open the file in Visual Studio.
     3. In the Build menu, select Build Solution.
     The application will be built in the default \bin or \bin\Debug directory.


To run the sample:
=================
     1. Navigate to the directory that contains the new executable, using the command prompt or Windows Explorer.
     2. Type DataGridViewCellIMEMode.exe at the command line, or double-click the icon for DataGridViewCellIMEMode.exe to launch it from Windows Explorer. 


Remarks
=================
     This sample works correctly only if Japanese IME is activated.
