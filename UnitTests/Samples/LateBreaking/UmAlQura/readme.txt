Um Al Qura Calendar Sample
==========================
     This sample uses the new System.Globalization.UmAlQuraCalendar class to display a calendar control  similar to the Month Calendar control.


Sample Language Implementations
===============================
     This sample is available in the following language implementations:
     C#
     Visual Basic


To build the sample using the command prompt:
=============================================
     1. Open the Command Prompt window and navigate to the VB subdirectory under the UmAlQuraSample directory.
     2. Type msbuild UmAlQuraSampleCS.sln or msbuild UmAlQuraSampleVB.sln.


To build the sample using Visual Studio:
=======================================
     1. Open Windows Explorer and navigate to one of the language-specific subdirectories under the UmAlQuraSample directory.
     2. Double-click the icon for the .sln (solution) file to open the file in Visual Studio.
     3. In the Build menu, select Build Solution.
     The application will be built in the default \bin or \bin\Debug directory.


To run the sample:
=================
     1. Navigate to the directory that contains the new executable, using the command prompt or Windows Explorer.
     2. Type UmAlQuraSample.exe at the command line, or double-click the icon for UmAlQuraSample.exe to launch it from Windows Explorer. 

Demonstrates
============
     This sample demonstrates the following:
     UmAlQuraCalendar, TableLayoutPanel, FlowLayoutPanel and RightToLeft
